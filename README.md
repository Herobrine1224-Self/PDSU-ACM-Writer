# 针对平顶山学院人工智能学院/软件学院[代码平台](http://39.106.228.241)进行开发

作为该平台的管理员，已有最高权限，Ctrl+C/V就能粘贴，但依旧对此非常痛恨，不想用但还不能不用

市面上有很多模拟键盘写入来绕过粘贴的工具，但在这个OJ上并不好用，因为OJ会补齐括号、会自动换行，所以，便有了此软件

一键启动，快速粘贴，支持C/C++/Java语言的提交，目前不支持Python的写入

基于Python+Webview快速开发，针对要粘贴的代码使用了神奇的算法，使其完美适配平台，写入速度尚可

当然也有其他方法，比如问我要个权限（显然行不通），工具可以分发，权限不行

如果感兴趣的话，可以研究研究ACE编辑器漏洞的[插件](https://github.com/wm-wu-min/pingyuan-oj-script)，这是另一个学长(对我来说其实并不是学长)开发的，使用浏览器插件解决问题，但我在此不赘述，那可能需要一些技术，显然不适合我们这些只会接受投喂的"宝宝"

如果你只是使用该工具，对其技术和原理不感兴趣，那大可不必下载源代码，已有封装好的exe，无需任何依赖，双击运行即可
